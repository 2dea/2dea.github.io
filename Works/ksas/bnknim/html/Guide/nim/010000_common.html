File Specifications

CSS(선언순서 정렬)

  • embed.css 웹폰트 정의
  • public.css 리셋 및 기본 타입셀렉터 정의, 공통레벨 스타일시트
  • plugins/swiper.css 슬라이드 플러그인 스타일시트 (2021-12-08 현재 미사용)
  • plugins/Chart.css 차트 플러그인 스타일시트
  • rewrite.css 플러그인 스타일시트 재정의 (VueFinalModal / RealGrid / MxDatepicker / Pagination 등)
  • module.css 컴포넌트 및 모듈 스타일 정의
  • layout.css 레이아웃 스타일 정의
  • function.css 애니메이션/트랜지션 및 동적 생성(JavaScript 등) 스타일 정의
  • index.css 로그인, 메인 등 주요 페이지 스타일 정의
  • content.css 부분적인 개별 페이지 스타일 정의
  • theme.css 다크 테마 등에 대응 스타일시트 (2021-12-08 현재 미사용)
  • common.css 스타일시트 임포트

Class-name Prefixes

  • g- (Global) 공통(public) 모듈 정의 { e.g. g-hidden(Screen-reader Only) 등 }
  • c- (Common) 프로젝트 공통(private) 모듈 정의 { 너비, 색상 등 }
  • o- (Object) 객체 단위 개별 공통 컴포넌트 { inline 객체 - 레이블, 버튼, 아이콘 등 }, { block 객체 - 섹션, 레이어 등 }
  • m- (Module) 영역 단위(대부분 Node를 포함) 공통 컴포넌트 { 테이블, 블릿 등 }
  • p- (Partial) 개별 페이지 단위의 모듈 컴포넌트
  • dom- (DOM) 레이아웃 요소 섹셔닝 그룹 정의
  • div- (DIVision) 섹셔닝 그룹내 그루핑 영역 정의
