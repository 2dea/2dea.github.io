Technology

세계 최초, 국내 최다 SDP 및 제로트러스트 기술 IPR 보유 PRIBIT만의 혁신적인 애플리케이션 접속 제어 기술을 만나보십시오.

Network Security Paradigm

TCP/IP 및 터널링 기반 네트워크 보안 솔루션이 도입된 곳 보다 도입되지 않은 곳을 찾기 힘듭니다.
보편화된 기술로 점철된 네트워크 보안 시장의 무한 경쟁 속에서 차세대 네트워크 보안 기술이 필요로 한 시점입니다.

기술이 발전할수록 통신 및 접속 대상은 점점 작아지고 있음.
  1. 1st Wave TCP/IP Firewall, UTM
    (Network Level)
  2. 2nd Wave Tunneling IPSec, SSL VPN
    (Device Level)
  3. 2.5th Wave Tunneling + IP SDP
    (Network + Device Level)
  4. What is 3rd Wave
비지니스 전환 필요 시점
  1. 포화 시장 (지각 수용 단계)
  2. 신규 시장 (선각 수용 단계)

Redefine the Micro Segment

지금까지 네트워크 접속의 단위는 단말로 생각되어 왔습니다.
하지만, 웹 브라우저, 문서 편집 도구, 파일 탐색기부터 바이러스, 멀웨어,
랜섬웨어까지 모든 애플리케이션은 통신을 합니다.
위험 애플리케이션 조차도 통신을 합니다.

이제 네트워크 접속은 단말보다 작은 애플리케이션을
제어할 수 있어야 합니다.

New Wave

차세대 네트워크 보안 요소인 애플리케이션 접속 제어 기술을 사용하면, 기존에 사용중인 애플리케이션 수정 없이,
비인가 비정상 애플리케이션의 접속을 원천 차단할 수 있습니다. 업무에 불필요로 한 앱, 백신으로 검출되지 않은 멀웨어,
랜섬웨어 등을 차단할 수 있는 가장 효과적인 방법을 제공합니다.

    1. 불특정
      단말로부터
      공격 가능
    2. 1st Wave TCP/IP Firewall, UTM
      (Network Level)

      애플리케이션으로부터

    3. 2.5th Wave Tunneling + IP SDP

      접속 제어 기술       SDP + α
      (Network + Device + Application)

New Standard

애플리케이션 접속 제어 기술은 클라우드에서 필요로 하는
애플리케이션만 손쉽게 접속을 On/Off 할 수 있습니다.
네트워크와 애플리케이션이 결합된 세계 최초의 알파 계층 네트워크
보안 기술은 네트워크 보안의 새로운 Paradigm을 제시합니다.

Turn Off Network, Identify Risks
단말의 네트워크 접속을 차단할 수 있습니다.
사용자의 네트워크 접속을 차단할 수 있습니다.
애플리케이션의 네트워크 접속을 차단할 수 있습니다.
위험이 발견되면 네트워크 접속을 차단할 수 있습니다.

IPR (Intellectual Property Resources)

네트워크 기술의 본 고장, 미국에서 인정받은 네트워크 보안 기술

PRIBIT은 혁신적인 R&D를 통해 차세대 네트워크 보안 및 글로벌 클라우드 보안 시장을 선도하고 있으며,
세계 최초 및 국내 최대 SDP 및 제로트러스트 아키텍처 기반 애플리케이션 접속 통제 관련 특허를
보유하고 있습니다.

Technology Maturity

PRIBIT만의 PCxF (PRIBIT Connect x Flow)는 전세계의 네트워크를 애플리케이션 수준에서 접속 제어 및
연결할 수 있는 세계 최초의 애플리케이션 전용 프로토콜로써 글로벌 네트워크 보안 기술 인수 및 다양한 IPR 확보,
다년간 R&D로 모방할 수 없는 독보적인 기술 성숙도를 보유하고 있습니다.

  1. 1st Wave Network Level
    • TCP/IP
  2. 2nd Wave Device Level (現 시장 눈높이)
    • On Demand Tunneling + Network (SDP)
    • Tunneling
  3. 3rd Wave Application Level (PCxF)
    • 10년 이상 사용할 수 있는 기술 및 IPR 확보 완료
    • 기술 개발까지 3년 소요 + α
    • 2030 + α
    • Flow Routing (Data Packet Control)
    • Application Authentication
    • TCP Session Control
    • On Demand Tunneling + Application + Network
