대출신청고객 성명 :

1. 대출모집인에 관한 사항
  • 가. 대출모집인이 전화, 문자메시지(SMS등), 이메일 등의 통신수단을 통해 연락했는지 여부
    예(             )             아니오
  • 나. 고객이 전화, 문자메시지(SMS등), 이메일 등의 통신수단을 통해 연락했는지 여부
    예(             )             아니오
  • 다. 대출모집인이 고객정보를 취득한 경로
    • ① 부동산 중개업소
    • ② 금융회사
    • ③ 제3자(               )
    • ④ 기타(                )
2. 고객에 대한 확인사항
  • 가. 고객 본인여부 확인(성명, 생년월일 등)
    예(             )             아니오
  • 나. 대출금액, 금리 , 만기 등 대출 사항을 정확히 인지하고 있는지 확인
    예(             )             아니오
  • 다. 대출모집인이 본인을 어떻게 소개했는지 여부
    • ① 은행직원
    • ② 대출모집인
    • ④ 기타(                )
  • 라. 대출모집인이 먼저 전화, SMS, ARS등을 통해서 고객에게 연락하였습니까?
    예(             )             아니오
  • 마. 고객님은 홈페이지 및 광고 (인터넷 또는 전단지 등) 매체를 통하여 대추롬집인에게 연락을 하였습니까?
    예(             )             아니오
  • 바. 고객님은 금융회사로부터 대출모집인을 소개받았습니까?
    예(             )             아니오
  • 사. 위의 내용이 아니라면 어떤 경로로 대출모집인을 알게되셨습니까?(구체적명시)
    예(             )             아니오
3.여신담당자의 점검결과 확인
  • 가. 대출모집경로의 적정성 확인여부(관련법규 준수 등)
  • 나. 불법정보 유통·활용행위 의심, 확인여부
  • 다. 불법정보 유통·활용행위 의심, 확인 시 조치내용

위 각 사항에 대한 기재사항은 사실과 다름없으며 본인이 직접 체크하고 작성하였음을 확인합니다.
